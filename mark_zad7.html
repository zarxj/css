Odpowiedzi na pytania

1. Na co pozwala styl lokalny, który element będzie formatowany z użyciem stylu lokalnego?

pozwala na nadanie unikalnego formatowania bezpośrednio wewnątrz znacznika HTML; formatowany będzie wyłącznie ten konkretny element, w którym atrybut został zapisany

2. Składnia teoretyczna (z użyciem ramki otaczającej może być tabela o jednej komórce, zastosuj „znaki specjalne” do napisania definicji stylu lokalnego tak, aby instrukcja CSS nie wykonywała się, a tylko wyświetlała wewnątrz ramki [  spacja niełamana] przykład „znaku specjalnego”)

Przykład definicji stylu lokalnego w CSS, pokazany jako tekst, aby instrukcja CSS nie była wykonywana:

                      <style> 

                                              body { 

                                                background-color: fuchsia; 

                                                font-size: 30px; 

                                                    } 

                      </style>

3. Co może być selektorem?

Selektorem w CSS może być:

  • nazwa tagu (np. body p
  • klasa (np. .klasa
  • identyfikator (np. #id)
  • atrybut (np. [type="text"])
  • pseudo-klasa (np. :hover)
  • pseudo-element (np. ::before)

4. Co wypisujemy jako "cechę". Jakie jest znaczenie średników?

Cechą jest nazwa parametru, który chcemy zmienić (np. color, margin), natomiast średnik pełni rolę separatora kończącego daną instrukcję, umożliwiając dopisanie kolejnych. Przykład cechy:

              <p style="color: blue; font-size: 20px;">Tekst</p>

Średnik (;) w CSS jest używany do oddzielania poszczególnych właściwości w bloku stylu. Oznacza to, że każda cecha powinna być zakończona średnikiem, z wyjątkiem ostatniej cechy w danym bloku stylu, choć jej obecność jest zalecana, aby uniknąć błędów przy dodawaniu nowych cech.

5. Co wypisujemy jako "wartość" (w deklaracji stylu)?

Wartością jest konkretne ustawienie dla danej cechy, czyli np. rozmiar (15px), kolor (red)
